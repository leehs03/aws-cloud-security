# ☁️ AWS 클라우드 보안 설정 및 점검 체크리스트

> SGA솔루션즈 TTS사업본부 지원 포트폴리오 | 컴퓨터정보학부 이형석

## 📋 프로젝트 개요

AWS 프리 티어 환경에서 클라우드 보안의 기본이 되는 설정(MFA, IAM 최소 권한, S3 퍼블릭 차단)을 직접 적용하고 점검 체크리스트를 제작한 프로젝트입니다.

## 🛠️ 사용 서비스

| 서비스 | 용도 |
|--------|------|
| AWS IAM | 계정 인증 및 권한 관리 |
| Amazon S3 | 데이터 스토리지 보안 설정 |

## 🔬 실습 내용

| 실습 | 내용 |
|------|------|
| 루트 계정 MFA 설정 | Google Authenticator 기반 2단계 인증 적용 |
| IAM 사용자 생성 | ReadOnlyAccess 정책으로 최소 권한 부여 |
| S3 버킷 보안 | 모든 퍼블릭 액세스 차단 및 검증 |
